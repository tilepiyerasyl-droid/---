# Консольное портфолио "Обо мне"

Это программа-портфолио, написанная на Python с использованием объектно-ориентированного программирования (ООП).

## Особенности проекта
* Данные читаются из файла `data.json`.
* Логика программы написана в классе `Portfolio`.
* Меню работает в бесконечном цикле `while`.

## Как запустить проект

1. Скачайте проект.
2. Убедитесь, что файлы `main.py` и `data.json` находятся в одной папке.
3. Откройте терминал в папке проекта и запустите команду:
```bash
python main.py
